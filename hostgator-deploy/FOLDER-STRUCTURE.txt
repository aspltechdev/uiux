PERMISSIONS GUIDE
====================================================================

Set These Permissions in HostGator:

Files (HTML, images, etc.):
  Permission: 644
  Meaning: Owner can read/write, others can only read
  
Folders (images/):
  Permission: 755
  Meaning: Owner can read/write/execute, others can read/execute

How to Set:
1. Select file/folder in File Manager
2. Click "Permissions" or right-click â†’ "Change Permissions"
3. Enter: 644 for files, 755 for folders
4. Click "Change Permissions"
